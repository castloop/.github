# castloop/.github

Org-level meta repository for [github.com/castloop](https://github.com/castloop). Holds canonical references that apply across every Castloop repo.

## What lives here

| Path | Purpose |
|---|---|
| [`profile/README.md`](./profile/README.md) | Shown on the org page at github.com/castloop |
| [`CONTRIBUTING.md`](./CONTRIBUTING.md) | Commit conventions, PR flow, branch naming — applies to all repos |
| [`STYLE.md`](./STYLE.md) | Brand voice rules for all written output (READMEs, docs, code comments, error messages) |
| [`CODE_OF_CONDUCT.md`](./CODE_OF_CONDUCT.md) | Contributor Covenant 2.1 |
| [`SECURITY.md`](./SECURITY.md) | Security disclosure policy |
| [`ISSUE_TEMPLATE/`](./ISSUE_TEMPLATE/) | Default issue templates inherited by every repo |

Per-repo `CLAUDE.md` files point here for the rules instead of duplicating them.
