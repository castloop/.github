# Castloop

**The runtime for living game characters.**

Castloop is open-source middleware that brings AI-driven characters and directors into shipping games. Sub-200ms dialog on-device. Persistent memory across sessions. World-canon-aware narrative consistency. One character config, native plugins for Unreal and Unity.

## What we're building

- **Characters that remember.** Persistent per-character memory that survives sessions and stays consistent with world canon.
- **Directors that adapt.** Runtime narrative orchestration in the lineage of Left 4 Dead's AI Director and Alien: Isolation's stalker AI.
- **Engine-native.** Unreal Engine plugin and Unity package, both calling into a shared Rust core via stable C ABI.
- **Local-first.** On-device inference for dialog and behavior. Cloud routing only when you opt into it.
- **Production-grade.** Built for shipping games, not demos — observability, cost caps, content validation, deterministic replay.

## Status

Early development. We're building in the open. Expect rough edges through 2026.
