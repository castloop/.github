# Security policy

Thank you for helping keep Castloop and its users safe.

## Reporting a vulnerability

**Do not open a public GitHub issue for security concerns.**

Email **security@castloop.dev** with:

- A description of the issue
- Steps to reproduce
- The repo and version affected
- Any suggested mitigation, if you have one

You will receive an acknowledgement within 72 hours. We will keep you informed as the issue is investigated and resolved.

## Disclosure

We follow coordinated disclosure. Once a fix is ready, we will:

1. Release a patched version
2. Credit you in the release notes (unless you prefer to remain anonymous)
3. Publish a security advisory on the affected repo

## Supported versions

Castloop is pre-release. Until v0.1.0, only the `main` branch of each repo is supported. After v0.1.0, the supported-versions table will be published here.
